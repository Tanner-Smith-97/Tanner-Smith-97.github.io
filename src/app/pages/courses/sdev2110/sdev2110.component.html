One of the assignments had me use an API to look up different movies based on there titles. I was then able to display information such as plot, and actors to the user. Towards the end of the course it covered things such as OOP(object oriented programming) which is something that I am familiar with from other courses I have taken. Along with how to store data locally to create little "save" states. through out the course I was able to learn about different UI and css options available to Javascript such as Bootstrap. Several of the assignments I build used boot strap to quickly add styles to the page.
